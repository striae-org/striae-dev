description: >-
  You can register and log in to Striae directly from your browser without
  Cloudflare WARP
visibility: PUBLIC
---
1. **Go to the Striae Authentication Page**

   * Navigate to [https://www.striae.org/auth](https://www.striae.org/auth)

2. **Enter Your Work Email Address**

   * On the page, enter your work email.

   * *Note: Your organization's domain must be registered with Striae for access.*

3. **Check Your Email for the Login Code**

   * Wait for a one-time code delivered to your inbox.

4. **Enter the Login Code**

   * Input the code on the Striae login screen to access the registration and login pages.

5. **Code Validity**

   * The code is valid for **one month**.

   * After one month, you will need to request and enter a new code.
